## Conclusion
AI or human estimates the percentage of AI (Artificial Intelligence) contributions in an image, the higher the value of AI percentage indicates the image is more likely to be generated by AI, and vice versa. This app is built to differentiate image made by AI and humans concerning an AI-generated image being contested in a human image-related art competition.
